AR-VIS Project

← Back to Portfolio

1. Overview

AR-VIS is a multi-user AR platform for interactively visualizing dynamic scientific data (e.g., magnetic fields), enabling embodied interaction overlaid on real-world space.

• Problem: Traditional data visualizations are mostly 2D and limited in leveraging users’ spatial and embodied cognition.

• Solution: To make “invisible” phenomena (like magnetic fields) tangible and manipulable in AR for scientists and learners.

[Image of augmented reality data visualization]

2. Objectives / Research Questions

  • Build a general-purpose AR visualization pipeline.
  • Support multi-user, embodied interaction with scientific data.
  • Demonstrate via dynamic phenomena (magnetic fields) over scales.

3. Methods & Implementation

  • Data pipeline & reduction: Methods for culling and formatting data for AR rendering.
  • Visualization/Rendering: Unity + Mixed Reality Toolkit + Hololens, shaders, and dynamic simulation.
  • Interaction & Multi-user support: Enabling multiple observers to share AR visualizations in real time.
  • Use case / demo: Visualizing magnetic fields from real sensor data (e.g., from NASA / magnetometers).

4. Key Contributions / Innovations

  • Embodied AR visualizations of invisible forces.
  • Real-time, networked multi-user AR environment.
  • Pipeline for mapping time-varying scientific data into tangible AR experiences.
  • Demonstration with magnetic field data to make abstract phenomena perceptible.

“AR-VIS demonstrates how AR can reframe how we perceive and interact with dynamic scientific data—making invisible forces tangible and shared in space. It bridges data science, spatial cognition, and embodied interaction.”
